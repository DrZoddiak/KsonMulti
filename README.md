# KsonMulti

How to add to your environment:

#Kotlin/JS: 

```
repositories {
    maven("https://repo.zodd.me/releases")
}

dependencies {
    implementation("me.zodd:KsonMulti-js:$version")
}

```
#JVM
```
repositories {
    maven("https://repo.zodd.me/releases")
}

dependencies {
    implementation("me.zodd:KsonMulti-jvm:$version")
}

```
